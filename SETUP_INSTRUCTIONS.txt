===============================================
TROUBLESHOOTING
===============================================

Problem: "Cannot find module 'express'"
Solution: Run "npm install" in the project folder

Problem: "Database connection error"
Solution: Check your DATABASE_URL in .env file is correct

Problem: "Port 8080 already in use"
Solution: Change PORT in .env to 8081 or another number

Problem: ".env file not found"
Solution: Make sure the file is named ".env" exactly (no .txt)
